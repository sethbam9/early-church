# Patristic harvesting operating rule

- Default mode sequence is Ask -> Plan -> Code.
- Never edit data files from a vague prompt.
- Always use the workflow files before making changes.
- Always update patristic-campaign-tracker.md during each batch.
- One batch at a time only.
- A batch is complete only when:
  1. evidence is collected,
  2. claims are written,
  3. uncertainty is marked,
  4. tracker is updated.
- Prefer primary sources and exact passage references.
- Do not infer beyond the evidence.
- When evidence is indirect, label it clearly as inference with stated certainty.
